# Future Internet source-format note - V15

The V15 manuscript is supplied as a self-contained free-format LaTeX source and audited review PDF. Future Internet permits free-format initial submission, provided the manuscript is complete and consistently formatted.

Before submission:

1. Reserve the version-specific Zenodo DOI and insert it in the Data Availability Statement.
2. Recompile and rerun the V15 audit after the DOI edit.
3. Verify the GitHub `v1.5.0` release and Zenodo record without authentication.

The manuscript may be transferred to MDPI's current official LaTeX class before submission or when requested by the journal. Any template conversion must preserve the audited content and be followed by a fresh compile, claim audit, and visual inspection. Official MDPI class files are not redistributed in this repository.
