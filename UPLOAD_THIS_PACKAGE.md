# GitHub upload package

This directory is the curated public repository for release `v1.5.0`.

Upload **all files and folders inside this directory** to the root of the GitHub repository. Do not upload the enclosing directory as an extra nested folder, and do not upload the ZIP as the only repository file.

The folders named `v7_*`, `v8_*`, `v9_*`, `v9b_*`, `v11_*`, `v12_exploratory`, `v13_confirmatory`, and `v15_seven_day_robustness` are retained because the final V15 manuscript reports results produced across those locked or archived study stages. They are reproducibility materials, not obsolete manuscript drafts.

Obsolete V11/V14 manuscript copies, the intermediate V14 audit, and duplicate submission files have been removed.
