# Contributing

Bug reports and reproducibility issues are welcome through GitHub Issues.
Please include the operating system, Python version, dependency versions, exact
command, and relevant log output.

Changes that alter numerical results should include:

1. a description of the modeling or implementation change;
2. a regression test or integrity check;
3. regenerated machine-readable summaries;
4. an explanation of any manuscript claim that changes.

Do not overwrite archived result files silently. Place revised campaigns in a
new versioned directory.
