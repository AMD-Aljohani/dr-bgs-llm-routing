# V15 submission checklist

1. Review and confirm the Author Contributions statement with both authors.
2. Create a Zenodo draft and reserve the version DOI.
3. Insert the reserved DOI into `manuscript/FutureInternet_manuscript_candidate_v15.tex`.
4. Publish the public GitHub tag `v1.5.0` using this exact package.
5. Publish the Zenodo record and verify the repository and DOI without signing in.
6. Recompile the manuscript and rerun `make audit-v15`.
7. Upload the compact submission ZIP to Future Internet.

Do not replace the locked V11, V12, V13, or V15 results during these administrative steps.
